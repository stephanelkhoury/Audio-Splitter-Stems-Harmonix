## 8. Technical Specifications

### 8.1 Image Optimization

| Format | Use Case | Quality | Max Size |
|--------|----------|---------|----------|
| SVG | Icons, illustrations, logos | N/A | <50 KB |
| PNG | Screenshots, transparency needed | Lossless | <500 KB |
| JPG | Photos, backgrounds | 80-90% | <300 KB |
| WebP | Web (with fallback) | 80-85% | <200 KB |

### 8.2 Responsive Images

Provide images at these breakpoints:
- Mobile: 640px wide
- Tablet: 1024px wide
- Desktop: 1920px wide
- Retina: 2x resolution

### 8.3 Color Profile

- Use sRGB color profile for all web images
- Ensure consistent appearance across devices

### 8.4 Naming Convention

```
{type}-{name}-{variant?}-{size?}.{format}

Examples:
- logo-primary.svg
- logo-primary-dark.svg
- hero-screenshot@2x.png
- icon-stem-vocals.svg
- og-image-twitter.png
```

- SVGO for SVG optimization
