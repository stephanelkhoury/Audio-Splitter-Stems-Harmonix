Backup Options

Full Backup

All data, configs, and user files

Creates a complete backup of the entire project including user data, configuration files, processed audio stems, and logs. Recommended for full system recovery.

Data Only

User accounts and settings

Exports only user data including accounts, preferences, contact messages, and configuration. Smaller file size, ideal for transferring user data.

Backup Information

  • Backups are created as ZIP files and downloaded to your device
  • Full backups include processed audio stems (may be large)
  • Store backups in a secure location
  • Automatic backups are created when stopping the server via ./scripts/stop.sh
